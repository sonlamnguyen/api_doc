This message uses for starting or stopping devices discovery feature on specified bridge. Response to this request with status devices_join message.

Topic

Format: lrn/hc/config
Example: lrn:luci:vn-hn:5/thing/99/hc/config

Parameters

Key Type Description
machc string HC to perform discovery action on, required.
brigde_key string Bridge to perform discovery action on, required.
action int Discovery action, valid value are 0 (stop) and 1 (start), required.
param object Information provided for Zigbee discovery session, optional.
value object Information provided for Zwave discovery session, optional.
